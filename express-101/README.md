# Express 101

### Project Creation Step by Step
- npm init -y
- npm i -D nodemon
- express install `npm i express`
- Nodemon `npm i -D nodemon` It is developer dependecny. 
- run project: `npm start` after setting nodemon then run `npm run dev`
