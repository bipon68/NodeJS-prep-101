# Raffle Draw Project

### Project Creation Step by Step
- npm init -y
- nodemon: `npm i -D nodemon`
- For ID - `npm i shortid`
- run project: `npm run dev`

**package.json file**

```node
 "scripts": {
    "start": "node src/server.js",
    "dev": "nodemon src/server.js"
  },
```
